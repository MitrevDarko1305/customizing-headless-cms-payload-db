
# Customizing-Headless-CMS-Payload-DB
This project customizes headless cms payload db, It includes custom collections, admin UI tweaks, and a tailored content structure to fit specific project needs.

# Features 
The CMS can serve a variety of content needs, with flexible components perfect for websites or applications. The dashboard and content structure are fully customizable to fit specific project requirements

# Setup (how to install it) 
1) Clone the repo to your local machine
2) In the terminal navigate to the project folder
3) Run npm install if needed, then npm run dev
4) Open your browser on localhost:3000
5) On the first run create a new admin user (email + password)

# Note - If you encounter missing dependencies or errors, ensure Node.js and npm are installed.
